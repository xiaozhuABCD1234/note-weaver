# Note Weaver

AI 驱动的 Obsidian 笔记助手 — 在编辑器内直接与 AI 对话，让 AI 读写、整理、检索你的所有笔记。

## 功能

| 特性 | 说明 |
| --- | --- |
| AI 聊天助手 | 侧边栏面板，支持流式输出 |
| 笔记工具调用 | AI 直接读取、创建、修改、删除、重命名笔记 |
| 知识检索 (RAG) | 自动检索 Vault 中与问题相关的笔记片段 |
| 网络搜索 | DuckDuckGo 搜索，获取实时信息 |
| 思考模式 | DeepSeek 等模型的内部推理，提升复杂任务能力 |
| Quick Ask | 编辑器内输入 `@` 触发内联问答 |
| 子 Agent | 复杂调研任务委派给子 Agent 并行执行 |
| 操作日志 | 记录 AI 交互、工具调用，便于调试 |

## 安装

1. 在 Obsidian 中打开 **设置 → 社区插件**，关闭安全模式
2. 点击 **浏览**，搜索 "Note Weaver"
3. 点击 **安装**，然后 **启用**
4. 或者手动将 `main.js`、`manifest.json`、`styles.css` 复制到 `<Vault>/.obsidian/plugins/note-weaver/`

## 配置

在 **设置 → Note Weaver** 中配置参数。

### LLM 设置

| 参数 | 说明 | 默认值 |
| --- | --- | --- |
| API Key | LLM API 密钥（支持 OpenAI 兼容接口） | — |
| Base URL | API 服务器地址 | `https://api.deepseek.com` |
| Model | 模型名称 | `deepseek-v4-flash` |
| Max Tokens | 每次请求的最大 Token 数 | `16384` |

### 思考模式

| 参数 | 说明 | 默认值 |
| --- | --- | --- |
| 启用思考模式 | 开启后模型先进行内部推理再输出（仅部分模型支持） | 开启 |
| 思考强度 | `high` 适合常规任务，`max` 适合复杂任务 | `high` |

### 网络搜索

| 参数 | 说明 | 默认值 |
| --- | --- | --- |
| 启用网络搜索 | 开启后 AI 可通过 DuckDuckGo 搜索互联网 | 开启 |
| 最大搜索结果数 | 每次搜索返回的最大结果数 | `5` |

### Quick Ask

| 参数 | 说明 | 默认值 |
| --- | --- | --- |
| 启用 Quick Ask | 开启后在编辑器输入 `@` 触发内联 AI 问答面板 | 开启 |

### RAG 知识检索

| 参数 | 说明 | 默认值 |
| --- | --- | --- |
| 启用 RAG | 开启后 AI 自动检索 Vault 中与问题相关的笔记 | 关闭 |
| 最大检索块数 | 每次查询最多检索的片段数 | `5` |
| 分块大小（字符） | 每个笔记片段的最大字符数 | `1000` |
| 分块重叠（字符） | 相邻片段之间的重叠字符数 | `200` |
| 检索范围 | 限制 AI 检索笔记的范围 | 整个 Vault |

## 工具能力

AI 助手可调用以下工具读写笔记和搜索内容：

| 工具 | 说明 |
| --- | --- |
| `read_note` | 读取指定路径的笔记完整内容 |
| `write_note` | 创建新笔记或覆盖已有笔记 |
| `append_note` | 向已有笔记末尾追加内容 |
| `edit_note` | 在笔记中做精确文本替换（查找并替换） |
| `delete_note` | 永久删除笔记（移至回收站） |
| `rename_note` | 重命名或移动笔记（自动更新内部链接） |
| `search_notes` | 按文件名或路径搜索笔记 |
| `search_content` | 在所有笔记中搜索文本内容 |
| `list_folder` | 列出文件夹内的文件和子文件夹 |
| `get_note_metadata` | 获取笔记元数据（标题、时间、标签、frontmatter） |
| `list_recent_notes` | 列出最近修改的笔记 |
| `web_search` | 通过 DuckDuckGo 搜索互联网 |
| `fetch_webpage` | 抓取网页可读文本内容 |
| `delegate_task` | 委派复杂任务给子 Agent 执行 |

## 使用

### 命令面板

| 命令 | 说明 |
| --- | --- |
| AI: 修改选中文本 | 将编辑器中选中的文本发送给 AI 进行处理 |
| 重建知识索引 | 重新扫描 Vault 构建 RAG 索引 |

### 聊天面板

点击左侧 Ribbon 栏的机器人图标或执行任意 AI 命令打开聊天面板。

- 输入消息按回车发送
- 按 Esc 取消正在进行的 AI 回复
- AI 自动调用工具读写笔记，操作后显示预览

### Quick Ask

在编辑器中输入 `@` 触发内联 AI 问答面板，输入问题后按回车获取回答，支持直接将回答插入到光标位置。

## 开发

```bash
# 安装依赖
npm install

# 开发模式（监听文件变化）
npm run dev

# 生产构建
npm run build

# 运行测试
npm run test

# 代码检查
npm run lint
```

构建产物为 `main.js`，复制到插件目录即可。
